h1. A port of Dojo's Deferred/Promise system to jQuery

It is based on the development version of Dojo which will become 1.5. See:
"Robust Promises with dojo.Deferred 1.5":http://www.sitepen.com/blog/2010/05/03/robust-promises-with-dojo-deferred-1-5/
for changes to the original dojo.Deferreds.

provides:

* jQuery.Deferred( )
* jQuery.DeferredList( )
* jQuery.when( )

and wrapper functions for jQuery's ajax functions:

* jQuery.d_ajax( )
* jQuery.d_get( )
* jQuery.d_getScript( )
* jQuery.d_getJSON( )
* jQuery.d_post( )

more documentation to come...
